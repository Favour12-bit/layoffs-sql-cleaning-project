# SQL Data Cleaning Project

## Project Overview
This project focuses on cleaning and standardizing a layoffs dataset using MySQL.  
The goal was to prepare raw data for analysis by removing duplicates, fixing blank values, standardizing text fields, and checking missing data.

## Tools Used
- MySQL
- MySQL Workbench
- SQL

## Cleaning Steps
- Removed duplicate rows
- Trimmed white spaces from company names
- Standardized industry names
- Filled missing industry values where possible
- Checked null and blank values
- Prepared the dataset for analysis

## SQL Skills Used
- SELECT statements
- UPDATE statements
- JOINs
- Self joins
- WHERE conditions
- GROUP BY
- HAVING
- TRIM()
- NULL handling

## Files
- `layoffs_data_cleaning.sql` — contains the SQL queries used for cleaning the dataset.
